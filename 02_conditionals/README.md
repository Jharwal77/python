# Python Conditionals Practice

A collection of beginner-friendly Python programs focused on mastering conditional statements (`if`, `elif`, `else`), logical operators, and decision-making in Python.

## 📚 Topics Covered

- `if`, `elif`, `else`
- Comparison operators
- Logical operators (`and`, `or`, `not`)
- Nested conditionals
- Ternary operator
- User input and type conversion
- Basic problem solving

## 📝 Problem Descriptions

### 01. Age Group Categorization
Categorize a person as Child, Teenager, Adult, or Senior based on age.

### 02. Movie Ticket Pricing
Determine ticket price according to age and apply discounts for special days.

### 03. Grade Calculator
Calculate grades based on marks entered by the user.

### 04. Fruit Ripeness Checker
Identify whether a fruit is Unripe, Ripe, or Overripe.

### 05. Weather Activity Suggestion
Suggest suitable activities depending on weather conditions.

### 06. Transportation Mode Selection
Recommend transportation methods based on distance and requirements.

### 07. Coffee Customization
Customize coffee orders using multiple user preferences.

### 08. Password Strength Checker
Evaluate password strength using conditional checks.

### 09. Leap Year Checker
Determine whether a given year is a leap year.

### 10. Pet Food Recommendation
Recommend food portions based on pet type and age.
