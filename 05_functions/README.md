# 📘 Python Functions

A collection of beginner-friendly Python exercises designed to build a strong understanding of functions and their various features. Each file focuses on a specific concept and contains the solution to a corresponding problem.

## 📂 Directory Structure

```
05_functions/
├── 01_square_function.py
├── 02_sum_two_numbers.py
├── 03_polymorphic_multiply.py
├── 04_circle_area_circumference.py
├── 05_default_parameter_greeting.py
├── 06_lambda_cube.py
├── 07_args_sum.py
├── 08_kwargs_printer.py
├── 09_even_number_generator.py
├── 10_recursive_factorial.py
└── README.md
```

## 🚀 Problems Covered

| # | Topic | Description |
|---|--------|-------------|
| 1 | Basic Function Syntax | Calculate and return the square of a number |
| 2 | Multiple Parameters | Return the sum of two numbers |
| 3 | Function Polymorphism | Multiply numbers or strings |
| 4 | Multiple Return Values | Find area and circumference of a circle |
| 5 | Default Parameters | Greet a user with an optional name |
| 6 | Lambda Functions | Compute the cube of a number |
| 7 | `*args` | Sum a variable number of arguments |
| 8 | `**kwargs` | Print keyword arguments as key-value pairs |
| 9 | Generators & `yield` | Generate even numbers up to a limit |
| 10 | Recursion | Calculate factorial recursively |

## 📚 Concepts Practiced

- Function definition with `def`
- Parameters and arguments
- Return values
- Polymorphism
- Multiple return values
- Default arguments
- Lambda expressions
- Variable-length arguments (`*args`)
- Keyword arguments (`**kwargs`)
- Generator functions and `yield`
- Recursion

## 🎯 Learning Objectives

By completing these exercises, you will learn how to:

- Write reusable functions.
- Pass and return data effectively.
- Use default and variable-length arguments.
- Create anonymous functions with `lambda`.
- Work with generators for efficient iteration.
- Understand recursive problem-solving techniques.

## ▶️ Running a File

```bash
python 01_square_function.py
```

Replace the filename with any exercise you want to run.
